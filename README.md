# adlibBot
 
USAGE:

Machine requirements: python and chrome

Mac/Unix:

- Open terminal
- `git clone https://github.com/jay6o/adlibBot.git`
- `cd /path/to/adlibBot/`
- `pip install -r requirements.txt`
- `python3 start.py`


Windows:
- Open Command Prompt
- `git clone https://github.com/jay6o/adlibBot.git`
- `cd \path\to\adlibBot\`
- `pip install -r requirements.txt`
- `python3 start.py`
